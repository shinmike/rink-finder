# Rink Finder (A Vue.js project)

Using API proivded by Hockey Community to show indoor and outdoor hockey rinks around Vancouver.

Created by Michael Shin

## Final Product

Main page
!["Screenshot of main page"](https://github.com/shinmike/rink-finder/blob/master/docs/rinkFinder.png)

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

For detailed explanation on how things work, consult the [docs for vue-loader](http://vuejs.github.io/vue-loader).
